== Description ==

The Payment Gateway SMBCGP for WooCommerce plugin adds the functionality to take SMBCGP payments on your store of WooCommerce.

= About SMBC GMO PAYMENT =

SMBC GMO PAYMENT is a comprehensive payment platform that offers various online payment methods, such as credit card payment and convenience store payment in Japan.

[SMBC GMO PAYMENT](https://www.smbc-gp.co.jp/)

[Privacy Policy](https://www.smbc-gp.co.jp/privacypolicy/index.html)

In order to use SMBCGP, you have to purchase the authentication key at the following site.

[WordPress Market](https://www.wpmarket.jp/)

== Installation ==

1. Upload the plugin file in the plugin install page or copy the `wc-smbcgp-gateway` directory into your `wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! :)
